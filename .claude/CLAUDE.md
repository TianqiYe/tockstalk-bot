# Global Claude Code Configuration

## Default Settings
- Model preference: claude-sonnet-4-20250514
- Hardware context: Available via @~/.claude/hardware-sdk.md
- Custom commands: Available in ~/.claude/commands/
- Sub-agents: Available in ~/.claude/config/agents/

Run `distiller-cc` in any project directory to set up local Claude Code environment with CM5-specific configuration.
